2.String上的操作

方法 参数 描述
length() String中字符的个数
charAt() int索引 取得String中该位置上的字符
getChars(), getBytes() 复制char或byte到一个目标数组中
toCharArray() 将String转换成一个char数组
equals(), equalsIgnoreCase() String 比较两个String内容是否相同
compareTo() String 按字典顺序比较两个字符串
contains() CharSequence 是否包含
contenEquals() CharSequence/StringBuffer 内容是否完全一致
equalsIgnoreCase() String 忽略大小写比较
regionMatcher() 偏移量, 比较对象及偏移量 检测两个字符串在一个区域内是否相等
startsWiht() String 是否以参数为起始
endsWith() String 是否以参数为后缀
indexOf(), lastIndexOf() 字符串 查索引值
substring() 起止索引 分割出新的字串
concat() String 拼接后的字符串
replace() 原串, 新串 替换字符串
toLowerCase(), toUpperCase() 转大小写
trim() 去两端空白
valueOf() … 获取值
intern() 返回字符串对象的规范化表示形式
